# Run
(1) Pull Docker Image
```
docker pull --platform linux/amd64 gs16100/team1_project:0.1.0
```
(2) run.sh
```
bash run.sh
```
(3) Web browser  
http://localhost:6080/vnc.html

# Game System
### (1) GUI
- GUI 시스템을 사용하여 구현    
- WASD 조작키로 이동 가능
- 대화창에서는 Y, N으로 선택 가능
### (2) 촌장과의 게임
- 촌장과 50G를 걸고 미니게임 가능
- 다양한 난이도의 미니게임 : 주사위, 홀짝맞추기
### (3) 계절 시스템
- 봄, 여름, 가을, 겨울 존재 : 30일이 지나면 계절이 바뀜
- 각 계절마다 재배가능한 작물이 다름
- 각 계절마다 랜덤 이벤트 존재 : 여름-태풍, 봄-메뚜기, 가을-화재, 겨울-폭설
    - 메뚜기 재해 발생시 모든 비축 식량이 사라짐 
### (4) 밭
- 5개의 작물 존재 : 밀, 쌀, 콩, 보리, 감자
- 각 작물마다 판매 가격이 다름 : 50원/100원/30원/20원/15원
### (5) 창고
- 창고에 작물 저장 가능 : 최대 저장 가능한 작물 수량 존재
- 골드로 업그레이드하여 창고 크기 늘릴 수 있음 : 100G, 1000G, 10000G
### (6) 시장
- 작물 가격은 계절마다 다르며, 매일 조금씩 변동 존재
- 봄에 재배되는 작물은 가을 겨울에 비쌈 (2배)
