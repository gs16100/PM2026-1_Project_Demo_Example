# Sprite Assets

PNG 파일 없이도 게임은 컬러 블록으로 실행됩니다.
스프라이트를 사용하려면 아래 파일을 이 폴더에 넣으세요.

| 파일명           | 설명                    | 권장 크기 |
|-----------------|------------------------|---------|
| ground.png      | 바닥 타일               | 64x64  |
| wall.png        | 벽 타일                 | 64x64  |
| player.png      | 플레이어 캐릭터           | 64x64  |
| home.png        | 집 (H 타일)             | 64x64  |
| market.png      | 시장 (M 타일)            | 64x64  |
| mayor.png       | 촌장 (G 타일)            | 64x64  |
| crop_stage0.png | 감자 - 씨앗 (성장도 0)    | 64x64  |
| crop_stage1.png | 감자 - 새싹 (성장도 1-2)  | 64x64  |
| crop_stage2.png | 감자 - 수확 가능          | 64x64  |
| livestock.png   | 닭                      | 64x64  |
| font.ttf        | 한국어 지원 폰트 (필수)    | -      |

## 무료 리소스 추천

- **폰트**: NanumGothic (`apt install fonts-nanum` 또는 https://hangeul.naver.com)
- **픽셀아트 타일**: https://itch.io 에서 "farming tileset" 검색 (무료 패키지 다수)
- **캐릭터**: https://opengameart.org 에서 "top down farmer" 검색
